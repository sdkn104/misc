## MCP server (Github copilot)

- install Github Copilot extension
- Choose "Agent mode" in chat window

- add settings in workspace settings.json
    - Ctrl + Shift + P（mac は Cmd + Shift + P）
    - コマンド検索欄に 「settings.json」 と入力
    - Preferences: Open Workspace Settings (JSON)
