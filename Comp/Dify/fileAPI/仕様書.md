

# RESTful File Storage API 仕様書

## 概要

ローカルファイルの管理（一覧取得・アップロード・ダウンロード・削除）を行うRESTful API。

* **Base URL**: `http://192.168.1.5:5050`
* **Version**: 1.0.0

---

## エンドポイント一覧

### 1. **GET /files**

ファイル一覧を取得する。

* **Response: 200 OK**

  ```json
  ["file1.txt", "image.png", "data.csv"]
  ```

---

### 2. **POST /files**

ファイルをアップロードする。

* **Request (multipart/form-data)**

  * `file` (binary, 必須)

* **Response: 201 Created**

  ```json
  {
    "fileId": "uploaded_filename.ext",
    "status": "uploaded"
  }
  ```

* **Response: 400 Bad Request**
  不正なリクエスト。

---

### 3. **GET /files/{fileId}**

指定ファイルをダウンロードする。

* **Path Parameter**

  * `fileId` : string（ファイル名）

* **Response: 200 OK**

  * Content-Type: `application/octet-stream`
  * バイナリファイルデータ

* **Response: 404 Not Found**

---

### 4. **DELETE /files/{fileId}**

指定ファイルを削除する。

* **Path Parameter**

  * `fileId` : string（ファイル名）

* **Response: 200 OK**

  ```json
  {
    "fileId": "example.txt",
    "status": "deleted"
  }
  ```

* **Response: 404 Not Found**
