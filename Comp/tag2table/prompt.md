
# 概要

- XMLファイルを読み込み、テーブル形式に変換してCSVファイルに出力する関数 convTag2TableFile
    - 引数は、XMLファイル名、CSVファイル名
- convTag2TableFileから呼び出す関数convTag2Table
    -　引数は、XMLテキスト変数、dataFrame
    - XMLテキストを読み込み、テーブル形式に変換してdataFrameに格納する

# 変換アルゴリズム

- テーブルの列名は、1,2,3,4,5,... とし、XMLの階層レベルを表す
- XMLタグをDFSで順番に捜査する
- XMLタグそれぞれについて、テーブルに値を書きだす
    - タグがテキストを持つときはテキストを書き出す、持たない場合はタグ名を「TAG:タグ名」の形で書き出す
- XMLタグは、そのタグの階層レベルの列に書き出す
- 最初のXMLタグは、一行目に書き出す。
- すでに同じ階層に値が書き出されている場合、行を一つ進めて書き出す
    - ただし、現在のタグがテキストを持たない場合は、行を進めず、また値を書き出さない
    - このとき、書き出した列より前の列については、前の行の値をそのままコピーする
